## Before Contributing

1. **Sign Legal Documents:**
   - Read and sign the [NDA Template](docs/NDA_TEMPLATE.md)
   - Read and sign the [Contributor Agreement](docs/CONTRIBUTOR_AGREEMENT.md)

2. **Understand Governance:**
   - Review [GOVERNANCE.md](GOVERNANCE.md)
   - All changes require petition-first approval
   - Badge system must be respected

3. **Review Guidelines:**
   - [SECURITY.md](SECURITY.md) - Security policies
   - [PRIVACY.md](PRIVACY.md) - Privacy requirements
   - [TERMS.md](TERMS.md) - Terms of service

## Development Workflow

1. Fork is **NOT allowed** - Request direct access
2. Create feature branch: `feature/your-feature-name`
3. Follow existing code patterns and Pi-only settlement rules
4. Submit PR with filled template
5. Wait for founder approval

## Code Standards

- Mobile-first responsive design
- TypeScript strict mode
- Pi Network integration only
- Badge-gated access
- Treasury transparency

## Compensation

Contributors receive civic shares or Pi-based rewards per governance decisions.
